# 实时聊天工具

基于UDP的实时通信工具

用户界面使用跨平台ui库[gio](https://github.com/gioui/gio)搭建  
通信模块使用[wi](https://github.com/CoyAce/wi)  

## 玩法

不需要账号就能使用，设置相同暗号的人可以一起聊天

## 设计原则

极简主义，keep it simple and stupid

# 想法

尽量使用自己写的代码，尽力重复造轮子，第三方库能不用就不用

不重复造轮子？前提是有造轮子的能力，不能和不想是两码事

首先解决能不能的问题，然后再解决想不想的问题

开发语言选择go主要是想学习go相关技术，go因为使用简单开发体检不错
